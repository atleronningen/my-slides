Implementeringsestimater

Aktiviteter og timeestimater

12 hovedaktiviteter fra AWS-fundament til pilot-onboarding. Blended rate kr 1 650/t. Alle estimater er angitt som intervall.

Aktivitet 01
AWS-fundament og Control Tower
Landing Zone, Control Tower, IAM Identity Center, Transit Gateway og Network Firewall. Kontostruktur for management, platform og kunde-sandboxer.
Estimat: 80–120 timer  ·  kr 132–198K
Aktivitet 02
Entra ID-integrasjon
Koble IAM Identity Center til kundens Entra ID via SCIM/SAML. Grupper, roller og tilgangsstyring per tenant.
Estimat: 20–40 timer  ·  kr 33–66K
Aktivitet 03
CI/CD-pipeline
GitHub-organisasjon, ECR-registre, GitHub Actions-workflows og CodePipeline for automatisk bygg og deploy av container-images per sandbox.
Estimat: 40–60 timer  ·  kr 66–99K
Aktivitet 04
Sandbox-template (VPC, ECS, RDS, WAF)
Standard VPC-oppsett med public/private subnets, ALB, WAF, ECS Fargate, RDS og Secrets Manager. Provisjoneres som CloudFormation/Terraform per tenant.
Estimat: 60–100 timer  ·  kr 99–165K
Aktivitet 05
AI Gateway og FinOps-instrumentering
Proxy-lag for Bedrock/Anthropic med tenant-ID-injeksjon, logging til S3 og Athena-queries for kostnadsallokering per app og tenant.
Estimat: 40–60 timer  ·  kr 66–99K
Aktivitet 06
Incubator Starter Template
GitHub-repo med Dockerfile, docker-compose, Actions-workflow og konfigurasjon som gjør en Vibecode-app Incubator-klar fra dag én.
Estimat: 20–30 timer  ·  kr 33–50K
Aktivitet 07
Deploy-portal (Vibecodervisning)
Webportal for app-deploy, statussider, logg-visning og kvotainformasjon. Entra ID-innlogging, URL-generering og onboarding-flyt med Starter Template.
Estimat: 80–120 timer  ·  kr 132–198K
Aktivitet 08
Beslutningstakerportal
Kostnadsdashbord, fakturaoversikt, kvote-godkjenningsflyt og brukeradministrasjon. Kobling mot CUR og AI Gateway-logger.
Estimat: 60–100 timer  ·  kr 99–165K
Aktivitet 09
Intern ops-flate (plattformteam)
Tenant-oversikt, FinOps-margin per tenant, WAR-statuskalender, plattformhelse og alerting. Kun tilgjengelig for 2S internt.
Estimat: 60–80 timer  ·  kr 99–132K
Aktivitet 10
Sikkerhet, WAR og compliance
Security Hub, GuardDuty, CloudTrail, backup-policy og WAR-gjennomgang. ISO 27001 / NIS2-kontroller dokumentert.
Estimat: 40–60 timer  ·  kr 66–99K
Aktivitet 11
Onboarding av første tenant
Full ende-til-ende-kjøring med reell kunde: provisjonering, Entra ID-kobling, første app deploy og fakturavalidering.
Estimat: 30–50 timer  ·  kr 50–83K
Aktivitet 12
Prosjektledelse og arkitektur
Teknisk ledelse, løpende arkitekturbeslutninger, dokumentasjon og koordinering gjennom hele leveransen.
Estimat: 40–60 timer  ·  kr 66–99K

Oppsummering

Nøkkelkonklusjoner

Plattformen er kommersielt bærekraftig med tre kunder og middels konsum. Engangs-investeringen amortiseres over 18 måneder. I tillegg kommer tilleggstjenester og konsulentoppdrag.

🏗
Byggekostnad er engangs
kr 1,2M er en strategisk investering, ikke løpende kostnad. Hver nye tenant etter de første tre går nesten rett til bunnlinje.
📈
Skaleringseffekt er stor
Plattformkostnaden er tilnærmet flat. En fjerde kunde i mnd 6 bidrar med kr 38 000/mnd ekstra og kutter break-even med 4–5 måneder.
🎯
Sokkel er kritisk
Fast sokkel på kr 28 000/mnd per kunde dekker løpende plattformkostnader uavhengig av konsum. Bør ikke forhandles ned.
